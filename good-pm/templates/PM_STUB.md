# Good PM Active

This project uses Good PM for spec/issue tracking.

## Quick Reference
- Status: Open (0/N) → In Progress (M/N) → Complete (N/N)
- Specs: `.good-pm/specs/SPEC_<name>.md`
- Issues: `.good-pm/issues/NNN-<desc>.md`

## Commands
- `/good-pm:issues` - View status summary
- `/good-pm:create-spec` - Create specification document
- `/good-pm:create-issues` - Break spec into issue files
- `/good-pm:setup` - Initialize Good PM in a project
- `/good-pm:help` - Detailed documentation and examples

## Full Documentation
PM commands auto-load full conventions when needed.
